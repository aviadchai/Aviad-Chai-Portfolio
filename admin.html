Connect to GitHub
The admin saves changes directly to your GitHub repo. Vercel deploys automatically after every save.
Create a GitHub Token
1. Go to github.com → your profile photo → Settings
2. Scroll to Developer settings (bottom left)
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Give it a name (e.g. "Portfolio Admin"), check repo scope
6. Click Generate token → copy it
