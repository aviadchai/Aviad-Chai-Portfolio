חיבור ל-GitHub
הפאנל שומר שינויים ישירות לריפו ב-GitHub. Vercel מפרסם אוטומטית לאחר כל שמירה.
יצירת GitHub Token
1. כנס ל-github.com → תמונת פרופיל → Settings
2. גלול ל-Developer settings (בתחתית)
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. תן שם (למשל "Portfolio Admin"), סמן public_repo
6. לחץ Generate token והעתק
